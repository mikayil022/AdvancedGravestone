# AdvancedGravestone

Advanced gravestone addon for Minecraft Bedrock Edition with item protection and advanced recovery features.

## Features
- Custom gravestone system
- Item protection on death
- Easy item recovery
- Bedrock Script API support
- Configurable behavior

## Installation
1. Download the `.mcaddon` file
2. Open it with Minecraft
3. Enable the addon in your world
